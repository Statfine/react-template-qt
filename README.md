# react_template

> React 项目模板

## Build Setup

```bash
# install dependencies
npm install

# serve with hot reload at localhost:3000
npm start

# creat page template
npm run generate

# build for production with minification
npm run build

# code standard
npm run lint

# run all tests
npm test
```
